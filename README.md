# Chip8emu
![alt text](https://github.com/Anh302kay/chip8emu/blob/main/icon.png?raw=true "Logo") \
Hobby Chip8 emulator with sound support.
Features:
- GUI ROM Loader
- Changeable draw colour
- Customisable scale and timescale
